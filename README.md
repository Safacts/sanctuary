# Sanctuary

A premium, ad-free mental wellness application designed for corporate professionals. Sanctuary provides a digital sanctuary through a sympathetic AI chat interface, grounding mini-games (like bubble wrap and a zen garden), and nervous system regulation tools.

## Features
- **Compassion Chat**: An empathetic AI-driven interface for venting and support.
- **Unwind Room**: 
  - **Digital Bubble Wrap**: Satisfying sensory interaction with procedural audio.
  - **Zen Garden**: Interactive sand raking for mindfulness.
- **Biometric Reset**: Box breathing (4-4-4-4 rhythm) with visual and audio cues.
- **Responsive Design**: Optimized for both Mobile (Bottom Nav) and Desktop (Sidebar).

## Deployment
This project is built using a "no-build" architecture (React + Tailwind via CDN) and is ready to be deployed on **Vercel** or any static hosting provider.

## How to Run Locally
Simply open `index.html` in any modern browser.
